## 功能描述

**自动生成数学作业，并保存为PDF。**

| 6  | 连加、连减     | a+b+c= 或 a-b-c=      | 30以内       |

### 解方程

`series = 'Form'`

| 等级 | 名称    | 公式                   | 说明        |
|----|-------|----------------------|-----------|
| 1  | 一元方程  | ax+b=c               | abc是整数    |
| 2  | 一元方程  | ax+b=c               | abc是分数    |
| 3  | 一元方程  | ax+b=c               | abc是小数    |
| 4  | 二元方程组 | ax+by=c, dx+ey=f     | abcdef是整数 |
| 5  | 二元方程组 | ax+by=c, dx+ey=f     | abcdef是分数 |
| 6  | 二元方程组 | ax+by=c, dx+ey=f     | abcdef是小数 |
| 7  | 二次展开  | (ax+b)^2=            | ab是整数     |
| 8  | 二次展开  | (ax+b)(ax-b)=        | ab是整数     |
| 9  | 二次展开  | (ax+b)(cx+d)=        | ab是整数     |
| 10 | 二次方程  | ax^2+b=c             | abc是整数    |
| 11 | 二次方程  | (ax+b)^2+c=d         | abcd是整数   |
| 12 | 二次方程  | e(ax+b)^2+c=d        | abcde是整数  |
| 13 | 二次方程  | (ax+b)^2+c=d         | abcd是分数   |
| 14 | 二次方程  | x^2+(b+c)x + bc = 0  | abc是整数    |
| 15 | 二次方程  | ax^2+(b+c)x + bc = 0 | abc是整数    |
| 16 | 二次方程  | ax^2+bx+c = 0        | ab是整数     |

## 使用方法

```python
from mathw import MathWork

if __name__ == '__main__':
    # 生成作业并保存成PDF
    MathWork(
        # Add - 加减法；Mult - 乘除法
        # Frac - 分数; Form - 解方程
        series='Form',
        level=4,  # 等级
        pageNum=20  # 页数
    ).go()
```
